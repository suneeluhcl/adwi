# Life Automation Ideas

Brainstorm of automation ideas across all life/work domains — not Adwi capabilities, but things that would make daily life noticeably better or faster.

For Adwi-specific capability ideas, see [[knowledge/Ideas Index]].
For scoring and lifecycle, see [[knowledge/Ideas Operating System]].

---

## Personal Productivity

- Morning brief auto-assembled from email + calendar + weather + top-3 tasks
- Smart "context switch" alert — detects when you've been in a rabbit hole for 90 min
- Weekly retrospective auto-draft from captured notes + git commits
- "What did I actually do today?" summary from activity log
- Smart clipboard history with semantic search

---

## Home / Phone Automation

- Home Assistant button → triggers Adwi task (already partially built)
- Phone shortcut to capture a voice note to Obsidian
- Door lock / unlock via Adwi command with confirmation gate
- Morning routine sequence: lights, thermostat, coffee, brief
- Away mode: one command sets lights + thermostat + security

---

## Email / Calendar

- Smart email digest: cluster threads by topic, surface only action items
- Calendar prep brief: 5 min before each meeting, relevant context from notes
- Auto-draft replies to common email patterns (with review-before-send gate)
- Unsubscribe radar: flag newsletters never opened in 30 days
- Meeting notes template auto-opened on meeting start

---

## Learning / Research

- Read-later queue with Adwi-powered summary on demand
- "What do I not understand?" gap detector from recent captures
- Spaced-repetition prompts from Obsidian notes via Adwi
- Auto-curate a "learn in 20 min" brief from bookmarks + Pocket
- Research thread: `/research` output auto-archived to `adwi/notes/research/`

---

## Health / Fitness

- Daily movement prompt if no activity captured by 2 PM
- Sleep consistency tracker — prompt if bedtime variance >45 min
- "No screen" reminder block (configurable hours, driven by Home Assistant)
- Water intake reminder (driven by phone shortcut or HA button)
- Weekly health summary assembled from captured fitness notes

---

## Finance / Admin

- Monthly spending summary assembled from bank transaction exports
- Recurring bill tracker — alert N days before due date
- Annual review: contracts, subscriptions, insurance — expiry reminders
- Tax prep assistant: gather receipts, categorize, produce a summary note
- Net worth snapshot — monthly, assembled from manual data entry prompts

---

## Coding / Workflows

- Automated "project status" note from git log + open PRs + test status
- Code review checklist auto-populated from diff analysis
- Dependency audit: weekly report of outdated packages across all projects
- PR description draft from commit messages + diff summary
- Dead code / unused file detector — weekly report

---

## Reminders / Follow-Ups

- Smart follow-up tracker: capture "follow up with X about Y by Z" in daily note → reminder fires
- Sent-email follow-up: flag emails with no reply after N days
- "Promised to do" tracker: capture commitments made in meetings
- Birthday / anniversary reminder with context note
- Recurring task generator: weekly / monthly tasks that auto-appear in daily notes

---

## Content / Reading / Video

- YouTube video → transcript → key points → Obsidian note
- Article → summary → capture to daily note with one command
- Podcast episode → show notes → searchable in vault
- Book reading tracker: current page, notes, key quotes captured via Adwi
- Tech radar digest: weekly summary of top HN / GitHub trending
