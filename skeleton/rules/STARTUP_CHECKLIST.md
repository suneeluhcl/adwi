# Startup Checklist

## Read First

1. `agent-system/shared/AGENT_SYSTEM.md`
2. `agent-system/shared/IDENTITY.md`
3. `agent-system/shared/WORKFLOW_RULES.md`
4. `agent-system/shared/SAFETY_BOUNDARIES.md`
5. `agent-system/shared/STARTUP_CHECKLIST.md`
6. `agent-system/memory/MEMORY.md`
7. `agent-system/memory/DECISIONS.md`
8. `agent-system/tasks/ACTIVE_TASKS.md`
9. `agent-system/tasks/TASK_QUEUE.md`
10. `agent-system/memory/SESSION_HANDOFF.md`
11. `agent-system/state/CURRENT_STATE.json`
12. `agent-system/state/WORKSPACE_HEALTH.json`

## Then Do

- Confirm the current user request.
- Run `agent-status` or `agent-doctor` when the workspace condition matters.
- Inspect relevant files before editing.
- Identify whether the work belongs in `projects/` or in the shared system.
- Avoid duplication.
- Preserve existing meaningful content with backups before replacement.
